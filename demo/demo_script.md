# Demo Script

## Goal
Show that the solution can predict attrition risk, explain the prediction, and audit fairness.

## Demo Flow

### 1. Business Problem
Introduce the HR issue:
- talent loss
- need for early detection
- need for responsible AI

### 2. Dataset
Show the variables used:
- salary
- department
- satisfaction
- absences
- engagement
- attrition target

### 3. Prediction
Present one employee example and display:
- predicted probability of attrition
- predicted class

### 4. Explanation
Explain the prediction:
- low satisfaction
- many absences
- short tenure
- repeated lateness

### 5. Fairness
Show the fairness audit by sex and/or race:
- compare recall
- compare false positive rate
- comment on any gap

### 6. HR Actionability
Conclude with practical actions:
- manager interview
- retention review
- mobility discussion
- workload / engagement follow-up
